4. 연산자(Operator)
연산자의 종류 : 산술연산자, 관계(비교)연산자, 논리연산자, 삼항연산자
(1) 산술연산자
    +  : 덧셈연산자
         str + str --> str들이 연결
    -  : 뺄셈연산자
    *  : 곱셈연산자
         str * int --> str이 int만큼 반복
    /  : 나눗셈연산자
         int / int --> float(소수)
    // : 몫 연산자
         10 // 3 = 3
    %  : 나머지연산자(모듈러스, modulo, mod)
         10 % 3 = 1
    ** : power(제곱) 연산자
         5 ** 2 = 25
         5 ** 3 = 125

(2) 관계(비교)연산자
    a > b   : a가 b보다 큰가?
    a >= b  : a가 b보다 같거나 큰가?
    a < b   : a가 b보다 작은가?
    a == b  : a와 b가 같은가?
    a != b  : a와 b가 다른가?

    관계연산자의 결과는 bool(True, False) 타입

(3) 논리연산자
    a and b : a가 True, b도 True면 True
    a or b  : a 또는 b가 True면 True (cf. 타언어의 XOR)
    not a   : a가 True면 False, False면 True

    논리연산자의 결과는 bool(True, False) 타입
